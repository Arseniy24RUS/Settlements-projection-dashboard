# Дашборд для GitHub Pages

Содержимое этой папки нужно разместить в корне репозитория GitHub Pages без изменения структуры каталогов.

## Состав

- `index.html` — основной интерфейс.
- `assets/` — стили и логика дашборда.
- `data/` — подготовленные Parquet-файлы и конфигурация.

## Публикация

1. Создайте репозиторий на GitHub.
2. Загрузите в него всё содержимое этой папки.
3. В настройках репозитория включите GitHub Pages и выберите публикацию из корня ветки.
4. После публикации откройте URL GitHub Pages.

## Важно

Дашборд рассчитан на работу по HTTP/HTTPS. Открытие файла `index.html` напрямую через `file://` может не дать доступа к данным.
